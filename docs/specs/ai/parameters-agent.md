# Parameters Agent Spec

## Purpose

Define prompts and context for the AI chat experience during initial model parameter setting.

## Initial Scope

- Help users select assets and describe modelling preferences.
- Support the preparation of an AI-generated modelling plan.
- Keep recommendations appropriate for an educational crypto treasury setting.
- Avoid presenting modelling suggestions as financial advice.

## Notes

- This spec should define system prompts, context inputs, allowed outputs, guardrails, and handoff to model plan confirmation after the initial prompt pass.
- This spec depends on the model parameters and AI model integration specs.
