# Riskfolio-Lib Spec

## Purpose

Define how Allocadabra uses `riskfolio-lib` to run supported models and generate outputs for users.

## Initial Scope

- Define the fixed set of models offered by the app.
- Define how selected models are run through `riskfolio-lib`.
- Define the range of model outputs generated by the app.
- Make outputs available for comparison, deeper review, AI reflection, and export.

## Notes

- This spec should stay light until the Modelling Agent has reviewed `riskfolio-lib` capabilities.
- This spec should eventually define supported model names, input dataset requirements, execution options, output artifacts, and failure modes.
